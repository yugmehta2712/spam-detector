# Spam Detector App

This is a simple machine learning-based web application that detects whether an email is spam or not using a trained model.

## Tech Stack:
- Python
- Flask
- Scikit-learn
- HTML (Frontend)

## How to Run:
1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
python app.py


3. Open in browser:
http://127.0.0.1:5000/
